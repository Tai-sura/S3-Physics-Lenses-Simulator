# 透镜光学模拟器

这是一个交互式透镜光学模拟器，用于演示凸透镜和凹透镜成像原理。模拟器使用HTML、CSS和JavaScript实现，无需任何外部依赖。

## 功能特点

1. **透镜类型选择**：
   - 凸透镜（产生实像或虚像）
   - 凹透镜（始终产生虚像）
2. **物体选择**：可选择学生或箭头作为观察物体
3. **参数调整**：
   - 物距调整（-30cm至30cm）
   - 焦距选择（3cm至15cm，凹透镜显示为负值）
   - 透镜位置调整（-15cm至+15cm）
4. **光线作图规则**：
   - 规则1：平行光线→经过焦点
   - 规则2：通过中心的光线
   - 规则3：经过焦点→平行光线
5. **测量数据显示**：
   - 物距
   - 像距
   - 放大率
   - 像类型（实像/虚像、放大/缩小、正立/倒立）
6. **视觉显示**：
   - 动态绘制光线路径
   - 物体和像的可视化
   - 网格背景
   - 焦点标记（F和2F点）

## 使用方法

1. 在浏览器中打开`index.html`文件
2. 使用控制面板上的选项和滑块调整参数
3. 点击透镜类型按钮切换凸透镜和凹透镜
4. 观察实时更新的像形成和光线路径
5. 勾选或取消勾选光线规则复选框，观察不同光线路径的作用

## 物理原理

该模拟器基于薄透镜方程和几何光学原理：

- **薄透镜方程**：1/f = 1/p + 1/q
  - f: 焦距（凸透镜为正值，凹透镜为负值）
  - p: 物距
  - q: 像距

- **放大率公式**：m = -q/p
  - m: 放大率
  - q: 像距
  - p: 物距

## 透镜特性

- **凸透镜**：
  - 可形成实像（倒立）或虚像（正立）
  - 实像位于透镜另一侧，虚像位于物体同侧
  - 放大率可大于1（放大）或小于1（缩小）

- **凹透镜**：
  - 始终形成正立的虚像
  - 像始终位于物体同侧
  - 放大率始终小于1（缩小）

## 文件结构

- `index.html` - 模拟器的HTML结构
- `styles.css` - 样式表
- `script.js` - JavaScript代码，包含所有计算和绘图逻辑
